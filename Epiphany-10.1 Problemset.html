A. Total Path
time limit per test
1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

You are standing at point (0,0)(0,0) in the 2-dimensional plane. Find the total number of ways to reach point (X,Y)(X,Y) using the minimum number of moves.

As it can be very large, so print it under modulo 109+7109+7.

If you are at the point (i,j)(i,j) in one move you can move to any one adjacent point out of 4.

  • (i−1,j)(i−1,j) : moving left
  • (i+1,j)(i+1,j) : moving right
  • (i,j−1)(i,j−1) : moving down
  • (i,j−1)(i,j−1) : moving up

Input

The input consists of multiple test cases. The first line contains an integer tt (1≤t≤1051≤t≤105) — the number of test cases. The description of the test cases follows.

The first and only line of each test case contains two integers XX (−106≤N≤106−106≤N≤106) and YY (−106≤M≤106−106≤M≤106)

Output

For each test case, Print one integer denoting the total number of ways to reach (X,Y)(X,Y) under modulo 109+7109+7

4
3 2
0 0
-4 -3
-1 2

input

standard output

Mike is given an undirected weighted tree, in which 1 is the root vertex. Control value of each vertex denoted by c1,c2,…,cnc1,c2,…,cn (where nn is the total number of vertices) is given to him. Let's take two vertices (V,U)(V,U) such that VV is the ancestor of UU, and VV controls UU if the distance between UU and VV is less than or equal to the control value of UU.

Note that here distance means the sum of the weights of the edges in the simple path between VV to UU.

Mike is asked to find the number of vertices control by each vertex. Can you help him.

Input

The first line contains single integer nn (1≤n≤2⋅1051≤n≤2⋅105).

The second line contains nn integers c1,c2,…,cnc1,c2,…,cn (1≤ci≤1091≤ci≤109) — the control value of ii-th vertices.

The next (n−1)(n−1) lines contain two integers each. The ii-th of these lines contains integers pipi and wiwi (1≤pi≤n1≤pi≤n, 1≤wi≤1091≤wi≤109) — the parent of the (i+1)(i+1)-th vertex in the tree and the weight of the edge between pipi and (i+1)(i+1).

Output

Print nn integers — the ii-th of these numbers should be equal to the number of vertices that the ii-th vertex controls.
